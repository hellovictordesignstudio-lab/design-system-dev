# Design System

> React 18 · TypeScript · Styled Components v6 · Storybook 8

Bienvenido a la documentación interactiva del Design System. Aquí puedes explorar todos los componentes, ver sus variantes, y copiar el código listo para usar.

---

## Cómo usar este sitio

- **Sidebar izquierda** — navega por componentes
- **Toolbar arriba** — cambia entre Light / Dark mode y ES / EN / FR
- **Docs tab** — ve la documentación y props de cada componente
- **Canvas tab** — interactúa con el componente en vivo
- **Controls panel** — modifica las props en tiempo real

## Tokens

El sistema usa tres capas de tokens:

**Primitivos** → valores crudos (`blue/500: #0055FF`)

**Semánticos** → con significado (`color/interactive/default`)

**Componentes** → usan semánticos, cambian automáticamente con el tema
